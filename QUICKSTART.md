# Quickstart Guide

## 1. Clone the Repository
```sh
git clone https://github.com/Suriya4169/eng2tam-unsloth-transformer.git
cd eng2tam-unsloth-transformer
```

## 2. Install Dependencies
```sh
pip install -r requirements.txt
```

## 3. Run the Notebook
Open `Notebook/eng2tam-unsloth.ipynb` in VS Code or Jupyter and run all cells to train and evaluate the model.

## 4. Output
- Trained model: `./tamil-translation-model`
- Evaluation report: `./evaluation_results.txt`

## 5. Customization
- Edit `src/config.py` or notebook config cell for hyperparameters.
